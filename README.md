
## General info
This project logins and shows userlist using reqresAPI, this component is accesed with token created after login, if you try to access to this component without login will redirect you to login page, you can delete any row of the table with delete button, and edit calling the API, it shows 6 users in the table but you can navigate to see the next or previous with the '<' and '>' buttons under the table.
